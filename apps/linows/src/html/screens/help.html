Keyboard Help Ctrl+H to close
Quick guide for app list, clipboard search, and command flow.
Main
EnterOpen selected app/file/folder or copy selected clipboard item
Ctrl+CCopy selected file/folder to clipboard
Ctrl+PToggle pick on selected file/folder (multi-select copy)
Ctrl+Shift+PClear all picked items
Tab / Shift+TabMove selection
Up / DownMove selection
Ctrl+FReveal selected item in file manager
Ctrl+EnterSearch current query on Google
Ctrl+/Enter command mode
Ctrl+Shift+,Open/close settings panel
Ctrl+Shift+;Reload .look.config
Ctrl+HToggle this help screen
EscClose help / back / hide launcher
Query prefixes
a"wordApps only
f"wordFiles only
d"wordFolders only
r"patternRegex search
c"wordClipboard history search (latest 10 text clips)
t"wordWeb translate (VI/EN/JA)
Command mode
Tab / Shift+TabSwitch command
Ctrl+1..5Switch to /calc, /pomo, /kill, /shell, /sys
:3000Find process listening on port
Up / DownSelect app in kill results
Y / NConfirm/cancel kill action
